# skia-pb

Minimal Skia build and distribution used by the Pb Noir project.

Contents
- `ios-release/`: prebuilt headers and libraries for iOS (include, lib, skia/arm64)
- `third_party/depot_tools/`: Chromium depot_tools used to build Skia

Quick start
- Use the provided build scripts to build or recreate the release artifacts:
	- `./build-skia.sh`
	- `./build-skia-ios-release.sh`
